## üé® **TECHNICAL DETAILS**

### **Radial Gradients**
```css
radial-gradient(circle, [color] 0%, [color] 40%, transparent 70%)
```
- Creates smooth circular gradients
- Fades to transparent at edges
- Multiple color stops for smooth transitions

### **Blur Filters**
```css
filter: blur(40px) to blur(80px)
```
- Heavy blur creates soft, diffused light
- GPU accelerated for performance
- Different amounts for depth

### **Motion Animations**
```typescript
animate={{
  x: [0, 150, 0],
  y: [0, 100, 0],
  scale: [1, 1.3, 1]
}}
```
- Arrays create keyframe animations
- Smooth easing with easeInOut
- Loop infinitely with `repeat: Infinity`

### **Spring Physics**
```typescript
transition={{
  type: 'spring',
  damping: 20,
  stiffness: 150
}}
```
- Natural, physics-based motion
- Smooth following behavior
- No overshooting
